# 不背药化 (NoMedChem) 💊

> 让药化不再难背——高效、可视化的药物化学智能辅助学习系统。

**不背药化** 是一个现代化的药物化学复习工具，专为药学专业学生设计。它结合了最新的前端技术与专业的药化数据，提供分子结构可视化、智能记忆曲线和多种学习模式，帮助你更高效地掌握复杂的药物化学知识。

## ✨ 核心功能

### 📚 五大学习模式

- **🎯 顺序学习**：循序渐进的记忆流程，包含四个阶段：辨识 → 熟悉 → 疗效 → 掌握。支持按章节筛选，可选顺序/乱序学习。
- **🧠 深度复习**：针对已掌握药物的强化复习，包含内部重学循环，系统会自动调整复习进度。
- **📖 药物目录**：按章节浏览全部药物，支持搜索功能和快速查看详情，可开关特定药物是否纳入学习。
- **📝 经典模式**：传统的卡片翻转模式，支持按章节筛选和特定类型筛选。
- **📋 模拟考试**：选择章节进行模拟测试，包含"写药名"和"写结构"两种题型。

### 🔬 专业切换

- **药化专业**：完整的药物化学课程内容
- **非药化专业**：针对非药化专业的精简版本

### 🎓 拟真考试模式

- 开启后分子结构图像变为**黑白**显示
- 支持 0°/90°/180°/270° **随机旋转**，模拟真实考试环境
- 预生成旋转图片，文字标签不会跟着旋转

### 📊 学习进度管理

- **智能抽认卡**：基于遗忘曲线的复习算法，自动安排学习计划
- **进度持久化**：使用 `localStorage` 自动保存学习状态
- **药物开关**：可在目录中禁用/启用特定药物，灵活调整学习范围

### 💡 其他特性

- **考点精要**：每个药物都配有详细的考点说明，涵盖结构、临床应用等信息
- **分子结构可视化**：预渲染的 SVG 分子结构图，原子按照标准配色（氧红、氮蓝、硫黄等）
- **键盘快捷键**：答题后按 `Space` 或 `Enter` 快速进入下一题
- **响应式设计**：完美适配桌面端与移动端

## 🛠 技术栈

- **核心框架**: [React](https://react.dev/)
- **构建工具**: [Vite](https://vitejs.dev/)
- **样式方案**: [TailwindCSS](https://tailwindcss.com/)
- **图标库**: [Phosphor Icons](https://phosphoricons.com/)
- **分子渲染**: RDKit (Python 脚本预生成 SVG 图片)

## 🚀 快速开始

如果你想在本地运行本项目：

1.  **克隆仓库**
    ```bash
    git clone https://github.com/Morphine-Hydrochloride/NoMedChem.git
    cd local_project
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **启动开发服务器**
    ```bash
    npm run dev
    ```

4.  **构建生产版本**
    ```bash
    npm run build
    ```

## ☁️ 部署

本项目支持通过 **Cloudflare Pages** 进行自动化部署：
1.  Fork 本仓库。
2.  在 Cloudflare Pages 中连接你的 GitHub 账号。
3.  选择本项目，构建命令设置为 `npm run build`，输出目录设置为 `dist`。
4.  点击部署即可。
