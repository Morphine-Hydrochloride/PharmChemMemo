以下整理出《药物化学》教学日历（第14章：作用于肾上腺素受体的药物）与教学PPT中均出现的药物。根据教学日历的要求（结构、理化性质、体内代谢、临床用途及合成方法），列出PPT中对应的知识点 。

### 重点掌握类药物（Master Category）

#### 1. 肾上腺素 (Epinephrine)

* 
**结构**：属于儿茶酚胺类神经递质 。


* 
**理化性质**：白色结晶性粉末，无臭，味苦；水中极微溶；具有酸碱两性，饱和水溶液显弱碱性；可溶于无机强酸和强碱，但不溶于氨和碳酸钠溶液 。


* 
**体内代谢/稳定性**：受光、热、金属离子影响不稳定；易发生氧化反应生成红色肾上腺素红，最终形成棕色多聚体 。


* 
**临床用途**：能兴奋心脏、收缩血管、松弛支气管平滑肌；临床用于过敏性休克、心搏骤停的急救，以及控制支气管哮喘的急性发作 。


* 
**合成方法**：PPT中没有列出 。



#### 2. 盐酸麻黄碱 (Ephedrine Hydrochloride)

* 
**结构**：(1R, 2S)-2-甲氨基-1-苯基-1-丙醇盐酸盐 。与肾上腺素相比，其苯环上无酚羟基，极性较小，易透过血脑屏障 。


* 
**理化性质**：白色针状结晶 。


* 
**体内代谢**：由于苯环上无酚羟基，不易被氧化和代谢，作用时间延长 。


* 
**临床用途**：用于支气管哮喘、过敏性反应及鼻黏膜充血肿胀引起的鼻塞；具有较强中枢兴奋性；可口服 。


* 
**合成方法**：PPT中没有列出 。



#### 3. 盐酸可乐定 (Clonidine Hydrochloride)

* 
**结构**：N-(2,6-二氯苯基)-4,5-二氢-1H-咪唑-2-胺盐酸盐 。


* 
**理化性质**：白色结晶性粉末，无臭；在水或乙醇中溶解 。


* 
**体内代谢**：口服迅速吸收；主要在肝脏进行羟基化（生成无降压活性的4-羟基可乐定）及与葡糖醛酸酯结合 。


* 
**临床用途**：中枢性降压药，选择性激动α2受体和I1-咪唑啉受体；原发性和继发性高血压均有效 。


* 
**合成方法**：以2,6-二氯苯胺为原料，经硫脲中间体、异硫脲中间体，最后与乙二胺反应并酸化成盐 。



#### 4. 盐酸多巴酚丁胺 (Dobutamine Hydrochloride)

* 
**结构**：(±)4-[2-[[1-甲基-3-(4-羟基苯基)丙基]氨基]乙基]-1,2-苯二酚盐酸盐 。


* 
**理化性质**：白色或类白色结晶性粉末，味微苦；在水或无水乙醇中略溶；游离碱遇光或空气颜色变深 。


* 
**体内代谢**：作用时间短，口服无效 。


* 
**临床用途**：选择性兴奋心脏β1受体，增加心肌收缩力；用于心力衰竭的治疗 。


* 
**合成方法**：PPT中没有列出 。



#### 5. 硫酸沙丁胺醇 (Salbutamol Sulfate)

* 
**结构**：不含儿茶酚胺结构 。


* 
**理化性质**：PPT中没有详细列出 。


* 
**体内代谢**：由于不含儿茶酚胺结构，可口服；约25%代谢形成硫酸酯 。


* 
**临床用途**：选择性激动支气管平滑肌β2受体，支气管舒张作用明显且持久；用于支气管哮喘、肺气肿等 。


* 
**合成方法**：PPT中没有列出 。



#### 6. 盐酸哌唑嗪 (Prazosin Hydrochloride)

* 
**结构**：含4-氨基-6,7-二甲氧基喹唑啉环和哌嗪环 。


* 
**理化性质**：类白色结晶性粉末；在乙醇中微溶，在水中几乎不溶；存在多晶型 。


* 
**体内代谢**：PPT中没有详细列出 。


* 
**临床用途**：选择性α1受体阻断剂；用于治疗高血压和充血性心力衰竭 。


* 
**合成方法**：PPT中没有列出 。



#### 7. 盐酸普萘洛尔 (Propranolol Hydrochloride)

* 
**结构**：1-异丙氨基-3-(1-萘氧基)-2-丙醇盐酸盐；属于芳氧丙醇胺类 。


* 
**理化性质**：白色或类白色结晶性粉末；无臭，味微甜后苦；在水或乙醇中溶解；对热和碱稳定，但对光和酸不稳定 。


* 
**体内代谢**：主要在肝脏代谢，产生萘酚和丙酸等代谢产物 。


* 
**临床用途**：非选择性β受体阻断剂；用于预防心绞痛，治疗心律失常 。


* 
**合成方法**：以1-萘酚为原料，与环氧氯丙烷反应生成环氧中间体，再与异丙胺反应并成盐 。



#### 8. 酒石酸美托洛尔 (Metoprolol Tartrate)

* 
**结构**：对位取代苯的芳氧丙醇胺类 。


* 
**理化性质**：又名倍他洛克，临床应用其酒石酸盐 。


* 
**体内代谢**：PPT中没有详细列出 。


* 
**临床用途**：选择性β1受体阻断剂；用于治疗心绞痛、心律失常和高血压 。


* 
**合成方法**：PPT中没有列出 。

### 熟悉类药物（Familiar Category）

#### 1. 去甲肾上腺素 (Norepinephrine/NA)

* 
**结构**：儿茶酚胺类神经递质 。


* 
**体内代谢**：体内生物合成从L-酪氨酸开始，经左旋多巴、多巴胺生成；代谢涉及COMT和MAO酶 。


* 
**临床用途**：对α1、α2和β1受体均有激动作用；收缩血管和升压作用强于肾上腺素 。



#### 2. 甲基多巴 (Methyldopa)

* 
**结构**：PPT列出了其S-(+)异构体结构 。


* 
**体内代谢**：前药，需经芳香AA脱羧酶和多巴胺-β-羟基化酶代谢成活性型α-甲基去甲肾上腺素 。


* 
**临床用途**：中枢性降压药，起效较慢但维持时间长 。



#### 3. 莫索尼定 (Moxonidine)

* 
**结构**：含有嘧啶环和咪唑啉环 。


* 
**临床用途**：主要激动I1受体，用于原发性高血压，中枢性不良反应少 。



#### 4. 盐酸异丙肾上腺素 (Isoprenaline)

* 
**结构**：儿茶酚胺N-取代基为异丙基 。


* 
**临床用途**：非选择性β受体激动剂，用于支气管哮喘急性发作 。



#### 5. 克仑特罗 (Clenbuterol)

* 
**结构**：苯环上有双氯取代和氨基 。


* 
**临床用途**：选择性β2受体激动剂，长效平喘药，可口服 。



#### 6. 特布他林 (Terbutaline)

* 
**结构**：间苯二酚结构变体 。


* 
**临床用途**：选择性β2受体激动剂，用于支气管哮喘、支气管痉挛 。



#### 7. 米拉贝隆 (Mirabegron)

* 
**结构**：PPT展示了其化学结构式 。


* 
**临床用途**：选择性β3受体激动剂；用于治疗膀胱过度活动症（OAB）引发的尿频、尿急 。



#### 8. 特拉唑嗪 (Terazosin)

* 
**结构**：将哌唑嗪结构中的呋喃环还原为四氢呋喃环 。


* 
**临床用途**：半衰期延长，毒性降低；除降压外，还用于治疗前列腺肥大引起的排尿困难 。



#### 9. 艾司洛尔 (Esmolol)

* 
**结构**：含有酯侧链 。


* 
**体内代谢**：极易被酯酶水解，血浆半衰期仅约10分钟，属超短效药物 。


* 
**临床用途**：控制房颤、房扑时的心室率，以及围手术期高血压 。
